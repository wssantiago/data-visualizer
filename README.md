# oncase-challenge
Oncase fullstack challenge

### dependencies
- React v18.2.0
- Node v16.14.2
- npm v8.11.0

### at the /oncase-challenge folder, start the project

```
cd client
npm install
cd ..
cd server
npm install
```

### launching the server

```
cd server
npm start
```

### launching the client

```
cd client
npm start
```
